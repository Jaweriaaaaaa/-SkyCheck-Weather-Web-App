SkyCheck — Weather Web Page

A real-time weather web page that shows live conditions for any city in the world, with dynamic animated backgrounds and weather-based advice notifications.

Built using vanilla HTML, CSS, and JavaScript.


Features


Search weather for any city worldwide
Real-time temperature, wind speed, and weather condition
Animated background that changes based on the weather (sun, clouds, rain, thunderstorm with lightning)
Animated rainbow border around the weather widget
Smart popup notifications with weather-based advice (e.g. stay hydrated on hot days, keep your umbrella on rainy days)
Responsive design



Built With


HTML5
CSS3 (animations, gradients, responsive design) Project Structure

index.html   — main page structure
style.css    — all styling and animations
script.js    — weather fetching logic and background animations
